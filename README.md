SUB-NET CALCULATOR JC 

Interactive subnet calculator built with Vanilla JavaScript. Calculates subnet ranges, hosts, CIDR, subnet masks, IP class/type, and visual bit distribution with color-coded network and host bits

DEMO 

https://josue-dev-0210.github.io/Calculadora-Subredes-JC/

FEATURES 

- Subnet calculation
- CIDR support
- Host range calculation
- Network and broadcast address
- Subnet mask convertion
- IP class detection
- Subnet class identification
- Visual bit distribution
- Color-coded network and host bits
- Responsive interface

TECHNOLOGIES USED 

- HTML5
- CSS3
- Vanilla JavaScript
